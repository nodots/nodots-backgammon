Quick Start

npm install @nodots-llc/backgammon-core
npm install @nodots-llc/backgammon-types  # Peer dependency
import { Board, Player, Game, Dice } from '@nodots-llc/backgammon-core'

// Initialize a new game
const board = Board.initialize()
const player1 = Player.initialize('black', 'clockwise')
const player2 = Player.initialize('white', 'counterclockwise')
const game = Game.initialize([player1, player2])

// Roll dice and make moves
const rolledGame = Game.roll(game)
const possibleMoves = Board.getPossibleMoves(board, player1, 6)

Core Modules

Board

Manages game board state, checker positions, and move validation.

Board.initialize() Creates new board with standard starting positions
Board.moveChecker() Executes checker moves with hit handling
Board.getPossibleMoves() Calculates all legal moves for a die value
Board.getPipCounts() Calculates pip counts for both players

Player

Manages player state, dice, and movement logic.

Player.initialize() Creates player with color and direction
Player.roll() Rolls dice for player in rolling state
Player.move() Executes moves from specific origins
Player.getHomeBoard() Gets home board points for player

Game

Orchestrates overall game flow and state management.

Game.initialize() Creates new game with two players
Game.rollForStart() Determines which player goes first
Game.roll() Rolls dice for the active player
Game.move() Executes moves in game context

Dice

Handles dice rolling and state management.

Dice.initialize() Creates new dice for a player
Dice.roll() Rolls dice and returns new state
Dice.isDouble() Checks if roll is a double
Dice.switchDice() Swaps the order of dice values

Type System

Core Types

BackgammonColor 'black' | 'white'
BackgammonMoveDirection 'clockwise' | 'counterclockwise'
BackgammonDieValue 1 | 2 | 3 | 4 | 5 | 6

Board Types

BackgammonBoard Main board interface with 24 points
BackgammonPoint Individual board positions (1-24)
BackgammonChecker Individual game pieces

Game States

BackgammonGame Discriminated union by state
BackgammonPlayer 9 different player states
BackgammonMove 3 move types: PointToPoint, BearOff, Reenter
